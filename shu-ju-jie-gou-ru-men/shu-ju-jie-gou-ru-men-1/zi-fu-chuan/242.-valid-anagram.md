# 242. Valid Anagram

排序 + 双指针

improve:sort后直接排序即可

```
class Solution {
    public boolean isAnagram(String s, String t) {

        if(s.length()!= t.length())
            return false;
        char [] news = s.toCharArray();
        char [] newt = t.toCharArray();
        
        Arrays.sort(news);
        Arrays.sort(newt);
        int j = 0;
        for(int i = 0; i < news.length;)
            {
                if(news[i] == newt[j])
                {
                    j++;
                    i++; 
                }
                    
                else 
                    return false;
            }
            return true;
    }
}
```
