# 383. Ransom Note

数组代替哈希

```
class Solution {
    public boolean canConstruct(String ransomNote, String magazine) {
        int a[] = new int[26];
        int b[] = new int[26];
        char aa[] = ransomNote.toCharArray();
        char bb[] = magazine.toCharArray();
        for(char c : aa) a[c-'a']++;
        for(char c : bb) b[c-'a']++;
        for(int i = 0;i < 26;i++){
            if(a[i] > b[i]) return false;
        }
        return true;
    }
}
```
