# 647. Palindromic Substrings

```
class Solution {
    int ans;
    public int countSubstrings(String s) {
        ans = 0;
        
        for(int i = 0; i < s.length(); i++){
            count(s, i, i);
            count(s, i, i+1);
        }
        
        return ans;
    }
    public void count(String s, int l, int r){
        while(l >= 0 && r <= s.length() -1){
            if(s.charAt(l) != s.charAt(r))
                break;
            l--;
            r++;
            ans++;
        }
    }
}
```
