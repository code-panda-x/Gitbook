# 300. Longest Increasing Subsequence

```
class Solution {
    public static int lengthOfLIS(int[] nums) {
        int[]dp = new int[nums.length];
        int max = Integer.MIN_VALUE;
        Arrays.fill(dp, 1);
        
        for(int i = 0; i < nums.length; i++){
            for(int j = 0; j < i; j++){
                if(nums[i] > nums[j])
                    dp[i] = Math.max(dp[i], dp[j]+1);
                
            }
            max = Math.max(max, dp[i]);
        }
        return max;
    }
}
```
