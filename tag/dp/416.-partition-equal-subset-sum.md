# 416. Partition Equal Subset Sum

Given an integer array `nums`, return `true` _if you can partition the array into two subsets such that the sum of the elements in both subsets is equal or_ `false` _otherwise_.

&#x20;

**Example 1:**

<pre><code><strong>Input: nums = [1,5,11,5]
</strong><strong>Output: true
</strong><strong>Explanation: The array can be partitioned as [1, 5, 5] and [11].
</strong></code></pre>

**Example 2:**

<pre><code><strong>Input: nums = [1,2,3,5]
</strong><strong>Output: false
</strong><strong>Explanation: The array cannot be partitioned into equal sum subsets.
</strong></code></pre>

class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for(int num : nums)
            sum += num;
        if(sum % 2 == 1)
            return false;
        sum = sum / 2;
        int n = nums.length;
        boolean[][] dp = new boolean[n+1][sum+1];
        
        for(int i = 0; i < n; i++)
            dp[i][0] = true;
        
        for(int i = 1; i <= n; i++){
            for(int j = 1; j <= sum; j++){
                if(j < nums[i-1])
                    dp[i][j] = dp[i-1][j];
                else{
                    dp[i][j] = dp[i-1][j] || dp[i-1][j-nums[i-1]];
                }
            }
        }
        return dp[n][sum];
    }
}

            sum += num;

            for(int j = W; j >= nums[i]; j--)
                dp[j] = dp[j] || dp[j-nums[i]];
