# 973. K Closest Points to Origin

```
class Solution {
    public int[][] kClosest(int[][] points, int k) {
        if(points.length == 0)
            return new int[0][];
        Arrays.sort(points, (a,b) -> a[0] * a[0] + a[1] * a[1] - b[0] * b[0] - b[1] * b[1]);
        return Arrays.copyOfRange(points, 0 , k);
    }
}
```
