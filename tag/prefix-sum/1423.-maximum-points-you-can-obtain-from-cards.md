# 1423. Maximum Points You Can Obtain from Cards

```
class Solution {
    public int maxScore(int[] cardPoints, int k) {
        int n = cardPoints.length;
        int lsum = 0;
        int rsum = 0;
        
        for(int i = 0; i < k; i++)
            lsum += cardPoints[i];
        int sum = lsum;
        
        for(int i = 0; i < k; i++){
            lsum -= cardPoints[k-i-1];
            rsum += cardPoints[n-i-1];
            sum = Math.max(sum, lsum+rsum);
        }
        return sum;
    }
}
```
