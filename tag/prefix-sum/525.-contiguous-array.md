# 525. Contiguous Array

```
class Solution {
    public int findMaxLength(int[] nums) {
        if(nums.length == 0 || nums == null)
            return 0;
        
        for(int i = 0; i < nums.length; i++)
            if(nums[i] == 0)
                nums[i] = -1;
        
        int sum = 0;
        int max = 0;
        HashMap<Integer, Integer> map = new HashMap<>();
        map.put(0, -1);
        
        for(int i = 0; i < nums.length; i++){
            sum += nums[i];
            
            if(map.containsKey(sum)){
                max = Math.max(i - map.get(sum), max);
                // 注意这里没有put map
            } else{
                map.put(sum, i);
            }
        }
        return max;
                
    }
}
```
