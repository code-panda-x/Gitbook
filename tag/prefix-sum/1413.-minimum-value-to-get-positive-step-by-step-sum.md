# 1413. Minimum Value to Get Positive Step by Step Sum

```
class Solution {
    public int minStartValue(int[] nums) {
        int sum = 0;
        int min = Integer.MAX_VALUE;
        for(int num : nums){
            sum += num;
            min = Math.min(sum ,min);
        }
        return min < 1 ? 1 - min : 1;    
        
    }
}
```
