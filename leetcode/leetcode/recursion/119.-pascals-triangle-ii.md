# 119. Pascal's Triangle II

每一行第一个始终为1，只需从后往前计算。

因为提前把每个值赋值为0，所以最后一个值总是 0 + 1 = 1

```
class Solution {
    public List<Integer> getRow(int k) {
        Integer[] arr = new Integer[k + 1];
        // k + 1 是因为 arr包含的是每一个row 的元素个数，第0row有1个，第1 row有2个元素，所以k+1
        Arrays.fill(arr, 0);
        arr[0] = 1;
        
        for (int i = 1; i <= k; i++) 
            for (int j = i; j > 0; j--) 
                arr[j] = arr[j] + arr[j - 1];
        
        return Arrays.asList(arr);
    }
}

Time: O N^2
Space: O k
```
