# 55. Jump Game

```
class Solution {
    public boolean canJump(int[] nums) {
        int max = 0;
        
        for(int i = 0; i <= max; i++)
        {
            max = Math.max(max, i + nums[i]);
            if(max >= nums.length)
                return true;
        }
        return false;
}
}


```
